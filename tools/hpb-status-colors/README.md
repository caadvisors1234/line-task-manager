# 既存タスクシートへの「タスク状況の色分け」組み込み手順

LINEタスク管理Botの「タスク状況をプルダウンで選ぶと行全体の色が変わる仕組み」を、既存の運用スプレッドシート(HPBタスクシート等)へ組み込むための単体スクリプトです。Bot本体とは独立しており、対象スプレッドシート側だけで完結します。

仕組みは**条件付き書式**(+任意でプルダウン)のみで、**セルの値には一切触れません**。既存の条件付き書式・プルダウンも消しません(下記の設定どおり)。

## 適用手順

1. 対象スプレッドシートを開き、メニューの **拡張機能 > Apps Script** を開く
2. エディタに `applyStatusColors.gs` の内容を貼り付けて保存する
3. ファイル冒頭の `SETTINGS` を対象シートに合わせて編集する
   - `TARGETS`: 対象タブごとに1エントリ
     - `sheetName`: タブ名(シート下部のタブに表示されている名前)
     - `statusColumn`: タスク状況が入っている列の列レター(例: `'H'`)
     - `firstDataRow`: データの開始行(1行目が見出しなら `2`)
     - `applyRange`: 色を塗る範囲(例: `'A2:R'`。開始行は `firstDataRow` と揃える)
   - `STATUS_STYLES` / `STATUS_ORDER`: 状況の値と色。既存シートの状況の表記に合わせて調整可(**表記が1文字でも違うと色が付かない**ため、実際のセルの値をコピーして合わせるのが確実)
4. エディタで関数 `applyStatusColors` を選択して実行する(初回はアカウントの承認ダイアログが出ます)
5. 実行ログに `[OK] タブ名: 既存ルール保全 n件 / 色ルール追加 9件 / プルダウン: …` が出ることを確認する
6. シートで状況の値を変えて、行の色が変わることを確認する

## 動作の要点

- **既存の条件付き書式は保全**: 追加するのは本スクリプトの色ルールのみ。再実行しても重複しません(自前ルールを張り替えるだけ)
- **プルダウンは既定で「未設定の場合のみ」設定**(`VALIDATION_MODE: 'missing-only'`)。既存のプルダウンがあれば触りません。色分けだけ入れたい場合は `'none'` に変更
- **やり直し**: 関数 `removeStatusColors` を実行すると、本スクリプトが追加した色ルールだけを除去します

## 制約(運用上の注意)

- `SETTINGS` の `statusColumn`・`firstDataRow`・状況の表記を**変更して再適用する場合は、先に変更前の設定のまま `removeStatusColors` を実行**してください(自前ルールの識別は現在の設定から生成した数式との一致で行うため、設定変更後だと旧ルールを除去できず残ります)
- `TARGETS` に**同じタブを複数エントリ書かない**でください(後のエントリが先のエントリの色ルールを除去し、最後の1エントリ分だけが残ります)
- プルダウンの「既存の設定があるか」は `firstDataRow` のセル1つで代表判定します。列の途中だけに別のプルダウンがある変則レイアウトでは、適用前に対象列の検証状態を確認してください
- 条件付き書式は**上にあるルールが優先**です。既存ルールを保全したうえで本スクリプトのルールを追加するため、同じ行に色を塗る既存ルールがあるとそちらが勝ち、状況の色が表示されないことがあります(その場合はシートの「表示形式 > 条件付き書式」でルールの並び順を調整してください)

## 色が付かないときの確認点

- 状況の値の**表記ゆれ**(全角/半角・中黒・スペースの有無)が `STATUS_STYLES` の `status` と一致しているか
- `statusColumn` / `firstDataRow` / `applyRange` が対象タブの実レイアウトと合っているか
- 「未対応」は白(既定色)のためルールがありません(仕様どおり)

## 色の定義(Bot本体と同一)

| タスク状況 | 行の色 |
|---|---|
| 未対応 | 白(既定) |
| 依頼中 | `#f4cccc` |
| 作業完了・未チェック | `#f4921e` |
| チェック完了・残りお客様連絡 | `#ec47dd`(文字白) |
| タスク完了済み | `#b7b7b7` |
| 佐藤さん提出 | `#ffe14d` |
| 急ぎの対応 | `#ff2b22`(文字白) |
| 反映待ち | `#4a86d8`(文字白) |
| お客様連絡待ち | `#b6d7a8` |
| 対象外 | `#d9d9d9`(取り消し線) |
